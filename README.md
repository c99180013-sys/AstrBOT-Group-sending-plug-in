# astrbot_plugin_group_sender

AstrBot 群发与定时发送插件。

## 功能

- 私聊机器人发送消息到指定群
- 私聊机器人群发到所有群
- 查看当前机器人所在群列表
- 支持每天定时发送
- 支持一次性定时发送
- 支持查看定时任务列表
- 支持取消定时任务

## 指令

### 立即发送

```text
/发送 列表
/发送 群发 内容
/发送 群号 内容
```

### 定时发送

```text
/定时发送 每天08:30 内容
/定时发送 群发 每天08:30 内容
/定时发送 123456789 每天08:30 内容

/定时发送 2026年04月02日20:30 内容
/定时发送 群发 2026年04月02日20:30 内容
/定时发送 123456789 2026年04月02日20:30 内容

/定时列表
/取消定时 编号
```

## 特性

- 管理员私聊控制
- 默认管理员 ID 已配置
- 定时任务本地持久化保存
- 适用于 AstrBot + aiocqhttp 环境

## 安装

方式一：AstrBot 插件页上传 zip 安装。

方式二：将插件目录放入 AstrBot 插件目录后重载。

## 文件说明

- `main.py` 插件主逻辑
- `metadata.yaml` 插件信息
- `README.txt` 简要说明

## 说明

定时任务数据默认保存在：

```text
data/plugin_data/astrbot_plugin_group_sender/schedules.json
```

## 适配

- AstrBot v4.x
- aiocqhttp
