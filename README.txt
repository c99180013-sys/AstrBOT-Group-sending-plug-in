AstrBot 群发插件（列表+定时版）
==============================

管理员ID已设置为：3827675264

支持命令：
1. /发送 列表
2. /发送 群发 内容
3. /发送 群号 内容
4. /定时发送 每天08:30 内容
5. /定时发送 群发 每天08:30 内容
6. /定时发送 123456789 每天08:30 内容
7. /定时发送 2026年04月02日20:30 内容
8. /定时列表
9. /取消定时 编号

说明：
- /定时发送 如果不写目标，默认是“群发”
- 一次性定时格式：XXXX年XX月XX日XX:XX
- 每天定时格式：每天XX:XX
- 定时任务数据保存在 data/plugin_data/astrbot_plugin_group_sender/schedules.json
